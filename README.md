# MyBadGirl 

MyBadGirl 是一个活动任务管理app，用户可以发布任务，查看任务，加入其他人创建的活动或者任务，任务设有统计跟排名，可以很直观的看到自己的状态，最近活跃情况，于好友私聊，建群群聊等等。

### 基本介绍

基于 angular2 + ui-boorstrap 练手demo,脚手架运用的是 angular-cli 。
